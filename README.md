# NEW Queue Monitor

Monitor the Debian FTP NEW queue with desktop notifications.

Built with GTK4/Adwaita. Part of the [Danne L10n Suite](https://github.com/yeager/debian-repo).

## Installation

### Debian/Ubuntu
```bash
sudo apt install new-queue-monitor
```
